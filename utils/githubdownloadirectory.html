GitHub Download Directory

This online utility allows downloading a GitHub directory, without downloading the entire repository. This is particularly useful for downloading a single simulation from the Java System Simulator Simulations repository.

The tool can be accessed here: https://download-directory.github.io/.

The most basic usage is to enter the URL associated with a folder from a GitHub repository. For advanced usage, it is possible to add a GitHub token, primarily useful to voercome rate limitations.

For other GitHub folder downloading tools, see this discussion.
